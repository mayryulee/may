# 카카오톡 세로형 미리보기 설정

카카오톡 **URL 붙여넣기**(링크 복사)는 카카오 서버가 이미지를 **가로 2:1(800×400)** 으로 잘라서 보여줍니다.  
**세로형 카드(1번 예시)** 는 **「카카오톡으로 청첩장 전하기」버튼 + 커스텀 템플릿**으로만 가능합니다.

## 1. 카카오 Developers 사전 설정

1. [Kakao Developers](https://developers.kakao.com) → 내 앱
2. **앱 > 제품 링크 관리 > 웹 도메인**에 등록:
   - `https://formayletter.netlify.app`
   - `http://localhost:5173` (로컬 테스트)
3. **앱 > 플랫폼 > Web** 사이트 도메인도 동일하게 등록

## 2. 메시지 템플릿 만들기

1. **도구 > 메시지 템플릿 > 템플릿 만들기**
2. **피드 A형** 선택
3. **사용 목적**: `카카오톡 공유`
4. **이미지** 설정:
   - 이미지: 스크랩 기본값 `${SCRAP_IMAGE}` 사용 (별도 사용자 인자 X)
   - **이미지 사이즈: 600 × 800** (3:4 세로)
   - **비율: Center Crop(0)**
5. **메시지 제목/본문**:
   - 제목: `${SCRAP_TITLE}` (또는 고정 문구)
   - 본문: `${SCRAP_DESCRIPTION}`
6. **버튼**:
   - 버튼명: `모바일 청첩장 보기`
   - 링크: `${SCRAP_REQUESTED_URL}` 또는 웹 도메인 `/`
7. 저장 후 **템플릿 ID** 복사 (숫자)

## 3. 환경 변수

`.env.local` (로컬) 및 Netlify 환경 변수:

```
VITE_KAKAO_SHARE_TEMPLATE_ID=123456
VITE_KAKAO_MAP_APP_KEY=발급받은_JavaScript_키
```

재배포 후 **카카오 공유 디버거**에서 캐시 초기화:
https://developers.kakao.com/tool/debugger/sharing

## 4. 테스트 방법

| 방법 | 미리보기 |
|------|----------|
| **노란 「카카오톡으로 청첩장 전하기」 버튼** | 세로형 카드 (600×800) |
| **링크 복사 후 붙여넣기** | 가로형(2:1) — 카카오 제한 |

세로형을 보려면 반드시 **공유 버튼**을 사용하세요.
